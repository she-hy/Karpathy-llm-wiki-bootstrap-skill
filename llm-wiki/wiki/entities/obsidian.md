**Obsidian** is a Markdown knowledge-base editor that the source frames as a practical browsing environment for an LLM Wiki.

## Role in the Source

The author describes using an LLM agent on one side and Obsidian on the other: the LLM edits pages while the human browses the updated wiki in real time. The source compares Obsidian to the IDE, the LLM to the programmer, and the wiki to the codebase.

## Useful Features Mentioned

- Web Clipper for turning articles into Markdown sources
- Graph view for seeing connections
- Attachment folder configuration for local images
- Plugins such as [[dataview|Dataview]] and [[marp|Marp]]
